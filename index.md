# 1-on-1 Kit
[English Version](index_EN.md) | [GitHub](https://github.com/stakiran/1on1-kit/settings/pages)

## はじめに

### 1on1 はつくれる
1on1 と聞いて何を思い浮かべますか？

おそらくトップダウンで指示されたものを浮かべるでしょう。たとえば「上司と部下が、月に一度行う、キャリアまたは仕事に関する対話」などです。

しかし 1on1 はこれに限らず、様々なあり方がありえます。何ならつくることができます。**必要な 1on1 を自らつくりましょう**。

### このキットについて
1on1 をつくるためのトリガーリスト（観点集）を揃えています。

※作者による参考実装であるため、各自カスタマイズしてください。参考実装では「テキスト完結の 1on1」すら行えるほどの柔軟性を確保しており、それゆえ項目が多くなっています。

## 1-on-1 Kit

```markdown
## 基本
- 頻度は？
    - 定期？週に一回、月に一回、隔週に一回 etc
    - 不定期？
- 時間は？
    - どれくらい？30分、45分、60分 etc
    - 半分経過する前に終えても良い？それとも頑張って粘る？
- 対象者は？
    - 誰と誰？
    - 同じレポートラインに限定する？しない？
    - 立場は変える？同じ？指定しない？
- 対称性は？
    - どちらがどれだけ喋る？1:9？3:7？5:5？

## 進行
- テーマの指定はある？
- ファシリテーションの有無は？有る場合、どっちがやる？

## 準備
- 開催様式は？対面会議？リモート会議？バーチャルオフィス？チャットやノートのみ？
- 申し込み方法は？また申し込む側はどっち？
    - 申し込みはどっちから始める？それとも自動化してソフトウェアに提案してもらう？

## 補助
- 録音の有無は？有る場合、自動で行える？それとも明示的な開始が必要？開始は誰がやる？
- 資料の有無は？有る場合、誰がどうやって共有する？また投影する？
- 1on1 の最中のメモは行ってもいい？
    - レベルは？1:禁止するか非推奨にする、2:禁止はしないが待たない、3:必要に応じて待つ、4:喋るよりも書く方が多い
```

## 例

### Free 1on1
全社員が誰でも誰にでもいつでも 1on1 できます。

- (基本)
    - 頻度: 不定期
    - 時間: 30分
    - 対象者: 社員全員
    - 対称性: 自由
- (進行)
    - テーマ: 自由
    - ファシリテーション: 自由
- (準備)
    - 開催様式: 自由。両者で合意すれば良い
    - 申し込み方法: 相手のグループスケジュールに勝手に予定を入れる
    - 申し込みを行う者: AさんがBさんと1on1したい場合、Aさん
- (補助)
    - 録音: 自由
    - 資料: 自由
    - メモのスタンス:
        - (*) 問わない
        - ( ) 禁止するか非推奨にする
        - ( ) 禁止はしないが待たない
        - ( ) 必要に応じて待つ
        - ( ) 喋るよりも書く方が多い

### Text 1on1
喋らずにテキストコミュニケーションのみで 1on1 をするというものです。

- (基本)
    - 頻度: 隔週に一回
    - 時間: 30分
    - 対象者: 同僚
    - 対称性: 5:5
- (進行)
    - テーマ: 任意
    - ファシリテーション: なし
- (準備)
    - 開催様式: テキストコミュニケーションのみ
    - 申し込み方法: 通常通り会議を開催する。また共同編集可能なページを一つつくる
    - 申し込みを行う者: ページの準備に慣れている側
- (補助)
    - 録音: なし
    - 資料: あり。ページ中に URL の形で付記しておく
    - メモのスタンス:
        - ( ) 問わない
        - ( ) 禁止するか非推奨にする
        - ( ) 禁止はしないが待たない
        - ( ) 必要に応じて待つ
        - (*) 喋るよりも書く方が多い
